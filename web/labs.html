Lab 2 – Advanced Layout

In this lab we used CSS to make the website responsive. This lab taught me a lot about the different features of CSS that allow me to change the way my website would look in different screen sizes.

What I found difficult was the balance between adding styles to the mobile website and the desktop website. Sometimes you could change something in the mobile version which has a negative impact on the desktop version.

I also used JQuery to manage my navigation bar. This is a new technique which is pretty handy.

The easy aspect of this lab was the form. I simply had to use all the different required form tags in my form.
